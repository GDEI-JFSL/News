# News

#### 项目介绍
微信小程序项目
涉及功能

高仿Readhub小程序

1.仿wx.showActionSheet组件， 因为wx.showActionSheet组件不可以调用分享，so自己手动实现

2.canvas文章内容自动换行，内边距功能实现 canvas绘图 

3.canvas动态生成图片功能  

4.canvas 自动适配 

5.文章左右滑动查看功能

6.canvas绘图时 ，目前有一个地方待优化，就当文章内容过多时，需要等比例缩放内容（解决中……）
